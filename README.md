# 🏢 CoWorking Space Management System

🚀 A complete Java-based Object-Oriented system that simulates a real-world coworking space management platform.

---

## 🎯 Overview

This system allows admins and visitors to manage rooms, reservations, and bookings efficiently.

---

## ⚙️ Features

### 👨‍💼 Admin

* Manage rooms and slots
* Add / Update / Delete reservations
* View revenue

### 👤 Visitor

* Register & Login
* Book and cancel slots
* Track usage

---

## 🛠️ Technologies

* Java
* OOP
* File Handling
